# Dropbox API
To run tests, execute the following command in the project directory:  
## Pytest
```
pytest
```
Expected output:
```
=== 3 passed in 4.18s ===
```
## Behave
```
behave
```
Expected output:
```
1 feature passed, 0 failed, 0 skipped
3 scenarios passed, 0 failed, 0 skipped
9 steps passed, 0 failed, 0 skipped, 0 undefined
Took 0m3.058s
```
